Netdiscover is active/passive arp reconnaisense tool to discover mac address of machine in local subnet.

Usage - netdiscover -i device -r netrange -p passive

-i = specify the network interface to be used. If not specified netdiscover will use fisrt interface.
-r = network range suppoerted (x.x.x.x/24, x.x.x.x/16, x.x.x.x/8)
-p = passive mode. In passive mode netdisvoer does not send/inject any arp packtes.

It will find only the mac address and ip of machines in local network ( same subnet).
