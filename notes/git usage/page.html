• Intialise local reporsitory
    • git init
  • Add file/file changes to local repo
    • git add -A
  • Commit changes to local repo
    • git commit -m "Notes Added"
  • Attach remote repo to local repo (First time only)
    • git remote add origin https://github.com/Dhanaraj-P/OSCP.git
  • Push changes from local to remote repo
    • git push -u origin master
  • Fetch changes from remote repo to local repo
    • git fetch origin
  • Merge the change from remote to local repo
    • git merge origin/master
  • Change git remote repo from https to SSH
    • git remote set-url origin <ssh_url>
